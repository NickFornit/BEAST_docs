Функция CheckCurActionsContext

Сохраняемый контекст ранее воздействующих акций

Какие сохраненные акции действуют в данный момент пульса - активные контексты действий с Пульта.
НЕ очищаются, сохраняя контекст до следующего Стимула.


Функция вызывается из:
  • automatizmTreeActivation - активация дерева автоматизмов
  • GetAttentionDanger - оценить опасность текущей ситуации: да-нет
  • GetCurrentInformationEnvironment - олучение текущего состояния информационной среды
  • GetCurrentConditionsStr - СТРОКА УСЛОВИЙ ДЛЯ безусловного РЕФЛЕКСА типа 1|2,5,8|11
