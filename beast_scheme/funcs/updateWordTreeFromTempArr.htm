Функция updateWordTreeFromTempArr

Переносим в дерево слов достаточно повторяющиеся

limitWord - число повторений отдельных слов, с которых начинается перенос.
limitFraze - число повторений отдельных фраз (с несколькими словами), с которых начинается перенос.
Для автозаливки - (4,6),
для сообщения с Пульта - (2,4)


Функция вызывается из:
  • SetNewTextBlock - Новая порция текста для формирования дерева слов
