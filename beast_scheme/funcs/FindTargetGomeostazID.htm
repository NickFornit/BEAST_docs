Функция FindTargetGomeostazID

Выявить ID параметров гомеостаза как цели для улучшения в данных условиях

Обеспечена сортировка по уменьшению важности: самые важные вначале. Это позволяет выбрать заданное число наиболее важных.


Функция вызывается из:
  • GetSelfPerceptionInfo - выдать самоощущение на Пульт
  • GetCurrentInformationEnvironment - получение текущего состояния информационной среды
  • GetActualReflexAction - сразу после активации дерева передать инфу для Психики
  • ChooseSimpleReflexexAction - выбрать подходящий простейший рефлекс и вернуть его действие
  • GetSimpleActionForCurContitions - выдать массив возможных действий по ID парамктров гомеостаза как цели для улучшения в данных условиях
