Функция createNewlastActivityID

Создать образ сочетаний пусковых стимулов, если такого еще нет

В случае отсутствия пусковых стимулов создается ID такого отсутствия, пример такой записи: 2|||0|0| - ID=2


Функция вызывается из:
  • automatizmTreeActivation - активация дерева автоматизмов
  • FindConditionsNode - поиск узла дерева автоматизмов по условиям у. рефлекса для automatizms_from_reflexes.go
  • CreateNewActivityImage - создать новый образ сочетаний действий, если такого еще нет
  • loadActivityFromIdArr - загрузить образы сочетаний пусковых стимулов
  • GetCurrentInformationEnvironment - получение текущего состояния информационной среды
